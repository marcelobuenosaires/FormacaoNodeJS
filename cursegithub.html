Curse basic de GITHUB

Date 18/04/2020

Comandos básicos: 

cd FormacaoNodeJS //* navegação da pasta do seu diretório na sua máquina

git clone https://github.com/marcelobuenosaires/FormacaoNodeJS.git //* Clonagem do diretório do seu projeto do git hub


git config --global user.email "marcelobuenos@yahoo.com.br" //* você esta configurando o email do seu git hub


git config --global user.name "marcelobuenosaires" //* Você esta configurando o usuario do seu git hub


git add README.md //* Você esta adicionando o arquivo READM.md para o repositorio do git hub

git commit -m "arquivo readme" //* " arquivo readme" você acrescenta o comentário do que você
fez ou esta subindo no repositorio.

git add. //* Manda todo o repositorio para o git hub

git commit -m "subindo arquivo index.html" //* Subindo arquivo index.html

git push origin master //* após o commit precisa enviar via upload para o repósitório o arquivo
